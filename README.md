# Product-Management-System
A full-stack application to manage products, including adding, updating, deleting, and viewing products. 
Built using Spring Boot for backend.

# Tech Stack
- Java
- Spring Boot
- MySQL
- Hibernate / JPA
- Maven

# Features
- Add / Edit / Delete Products
- View Products in list
- REST API for product management
- Basic Authentication (Optional)

## How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run Application.java (Spring Boot)
